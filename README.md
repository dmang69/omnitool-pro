# AI Toolkit

A modular Windows-friendly AI developer toolkit with:
- CLI chat interface
- plugin system
- safe file system tools
- Python code understanding
- controlled automation
- authorized Android/iPhone diagnostics
- official factory reset / restore plans
- plugin-based tool scaffolding

## Important
This project intentionally does **not** include:
- FRP bypass
- iCloud / Activation Lock bypass
- IMEI blacklist removal
- Knox / MDM bypass
- Rubber Ducky / O.MG payload tooling
- lock circumvention or anti-theft evasion
